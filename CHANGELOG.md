# Changelog

All notable changes to `larapdfx` will be documented in this file.

## v1.0.0 - 2025-01-19

### Added
- Initial release
- Full CSS3 support (Flexbox, Grid, modern properties)
- Chrome headless rendering engine integration
- Multiple output formats (download, stream, save, base64)
- Paper format support (A4, A3, A5, Letter, Legal, Tabloid, Ledger)
- Custom paper sizes
- Landscape and portrait orientation
- Configurable margins
- Headers and footers with page numbers
- Password protection (user and owner passwords)
- PDF permissions (printing, copying, modifying)
- Text watermarks with customization
- PDF metadata (title, author, subject, keywords, creator)
- RTL language support
- Custom fonts support
- Background printing control
- Scale factor control
- Timeout configuration
- Custom Chrome/Node.js paths
- Artisan commands (`larapdfx:install`, `larapdfx:test`)
- Comprehensive documentation
- Production-ready error handling
- Laravel 10 and 11 support
- PHP 8.1+ support
